# iWiFi
共享Wifi项目

**该项目仅提供学习，未经许可禁止用于其他用途**

## 1. 项目介绍

iWiFi是一款基于地理位置定位的手机实时连接周围WIFI的应用软件。有多余流量或宽带用户可以通过共享WIFI将WIFI信号共享出来，人们可以通过共享WIFI的地图查看周围有哪些可以连接的WIFI，系统将自动计算流量的消耗并进行双方的结算。在共享经济盛行的当下，将WiFi共享出来，能最大化的利用宽带、流量资源，实现分享者、消费者、平台三方共赢。

## 2. 产品介绍
### 2.1 主要服务

使用本平台，用户可以随时随地打开APP，自动获取附近用户开启的手机便携式热点或者宽带热点，以远低于运营商流量资费的价格按使用量计费。当用户流量有多余时，可以通过本平台进行热点共享，获取一定的收益，用户也可以将宽带热点进行共享，同样可以获得巨大的收益。

### 2.2 主要功能
- **连接共享WiFi**
通过我们的APP平台，用户走到何处都可以打开APP，自动获取周围其他用户开启的手机便携热点或者固定宽带热点，获取热点信息连接上后，即可以远低于运营商流量资费的价格按使用量计费。主页面可实时显示流量使用量和需支付费用，也可设置上限提醒，让用户能够合理使用。也不必担心wifi安全问题，我们提供了安全检测，杜绝钓鱼热点，对平台分享的非正常wifi，将对其发布者将进行信用分扣减。

- **信号增强、安全检查**
在连接WiFi过程中，app会自动选择最优配置进行连接，并对此次WiFi连接进行安全检查，避免钓鱼WiFi的风险。

- **分享热点**
当用户手机流量剩余较多或周围其他人继续网络连接，可开启并配置热点信息，勾选上共享选项，平台会将热点信息以及分享者的位置信息实时上传更新到云端，附近其他人可以搜索到，也可以在热点地图中查看该热点位置和其他一些信息。当有设备连接到分享的热点，热点主人可以查看到该设备，以及实时统计得到的流量消耗量和收益值。在使用者定制使用或分享者自行关闭热点后，分享者将获得相应金额收益并存放于钱包中。

<img src="https://github.com/biloba123/WifiSharing2/blob/master/images/%E5%88%86%E4%BA%ABWiFi_2.png" width="40%">

- **热点地图**
当用户想要连接共享wifi，然而身边有没有时，可以打开热点地图，获取周围分享的热点，并在地图中相应位置标注出来。其中蓝色为固定宽带热点，同时记录访问人次，绿色为周围其他用户开启的移动便携热点，其位置信息每隔一定时间会进行更新。点击地图上的标记，即可查看热点相关信息，具体位置，与当前位置距离，也可以直接通过导航，app规划路径引导用户前往相应地点。

- **个人信息**
在用户个人信息界面，提供了个人钱包，用户可进行充值、提现等操作，钱包的引入避免了共享wifi使用者和分享者进行第三方支付调用的繁琐过程，直接对钱包中余额进行操作。在我的共享中，用户能查看管理自己所分享的固定宽带热点，它们的访问量，带来的收益等信息。平台同时也引入一套完整的用户信用记录体系，对用户违反规范的不良行为进行扣除相应量的信用分，而每次完成一次分享能加分，信用分的高低对其他用户可见，而且相应信用分能得到相应的限制或特权，以规范平台整体秩序正常运行。
